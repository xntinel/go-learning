# 9. Building a CLI Code Generator

A production-quality CLI code generator combines the three skills from the preceding lessons: AST parsing to discover types, template rendering to emit code, and a robust CLI interface that handles flags, errors, and multiple output modes. Tools like `mockgen`, `protoc-gen-go`, and `ent` all follow this architecture. Building your own forces you to make every architectural decision they made: what constitutes the model layer, how to handle multi-type inputs, when to fail hard versus warn and continue.

```text
gogen/
  go.mod
  gogen.go
  gogen_gen.go
  gogen_test.go
  cmd/gogen/main.go
```

The library layer owns parsing and generation. The CLI layer owns flag parsing, I/O, and exit codes. Tests exercise the library layer directly in-process, without shelling out.

## Concepts

### Separating Parser, Model, and Generator

The cleanest code generators have three layers:

1. **Parser**: reads Go source bytes, returns a slice of `StructInfo` values (pure extraction, no rendering).
2. **Model**: typed data structures (`StructInfo`, `FieldInfo`) that carry everything the generator needs. No AST types leak into this layer.
3. **Generator**: takes a package name and a slice of `StructInfo`, renders a template, formats the output, returns `[]byte`.

This separation means:
- The parser is testable without a template.
- The generator is testable without a filesystem.
- The CLI is a thin orchestrator that can be replaced independently.

### Multiple Generation Modes

A real generator supports multiple modes via a flag or subcommand. Each mode is a function that accepts `[]StructInfo` and a package name, and returns `([]byte, error)`. The CLI dispatches to the right function based on the `-mode` flag.

### Type Filtering

When `-type=User,Config` is specified, the CLI filters the parsed structs to the named subset. Unknown names produce a warning on stderr but do not abort the run — a real generator may have a partial type list during development. Only exit if the matched set is empty.

### Required-Field Discovery via Struct Tags

Use `reflect.StructTag` on the raw AST tag value (backticks stripped) to check for `validate:"required"`. The same pattern used for JSON tags applies:

```go
raw := strings.Trim(field.Tag.Value, "`")
if reflect.StructTag(raw).Get("validate") == "required" {
    fi.Required = true
}
```

### Generating Validators That Collect Errors

A validator that returns on the first error forces the caller to fix issues one at a time. Collecting all errors into a slice and joining them in one return is more useful in practice:

```go
var errs []string
if s.Name == "" {
    errs = append(errs, "name is required")
}
if len(errs) > 0 {
    return fmt.Errorf("validation failed: %s", strings.Join(errs, "; "))
}
return nil
```

### `go/format.Source` and the `DO NOT EDIT` Header

Every generated file must begin with `// Code generated by gogen; DO NOT EDIT.` before the `package` declaration. `go/format.Source` verifies syntactic correctness. Always print the raw template output to stderr when `format.Source` fails — the error alone is rarely enough to locate the mistake.

### Exercise 1: Model and parser

Create `gogen.go` with the shared model types and the parser:

```go
package gogen

import (
	"fmt"
	"go/ast"
	"go/parser"
	"go/token"
	"reflect"
	"strings"
)

// FieldInfo holds metadata for one exported struct field.
type FieldInfo struct {
	Name     string
	Type     string
	JSONName string
	Required bool
	IsPtr    bool
}

// StructInfo holds the parsed metadata for one exported struct type.
type StructInfo struct {
	Name   string
	Fields []FieldInfo
}

// ParseStructs parses Go source bytes and returns metadata for each exported struct.
// Unexported structs are skipped. Unexported fields within exported structs are skipped.
func ParseStructs(src []byte) ([]StructInfo, error) {
	fset := token.NewFileSet()
	f, err := parser.ParseFile(fset, "input.go", src, 0)
	if err != nil {
		return nil, fmt.Errorf("parse: %w", err)
	}

	var structs []StructInfo

	ast.Inspect(f, func(n ast.Node) bool {
		ts, ok := n.(*ast.TypeSpec)
		if !ok {
			return true
		}
		if !ast.IsExported(ts.Name.Name) {
			return true
		}
		st, ok := ts.Type.(*ast.StructType)
		if !ok {
			return true
		}

		si := StructInfo{Name: ts.Name.Name}
		for _, field := range st.Fields.List {
			if len(field.Names) == 0 {
				continue
			}
			if !ast.IsExported(field.Names[0].Name) {
				continue
			}

			_, isPtr := field.Type.(*ast.StarExpr)
			fi := FieldInfo{
				Name:     field.Names[0].Name,
				Type:     exprString(field.Type),
				Required: false,
				IsPtr:    isPtr,
			}

			if field.Tag != nil {
				raw := strings.Trim(field.Tag.Value, "`")
				tag := reflect.StructTag(raw)

				jsonVal := tag.Get("json")
				if jsonVal != "" && jsonVal != "-" {
					parts := strings.SplitN(jsonVal, ",", 2)
					fi.JSONName = parts[0]
				}
				if fi.JSONName == "" {
					fi.JSONName = strings.ToLower(fi.Name)
				}

				if tag.Get("validate") == "required" {
					fi.Required = true
				}
			} else {
				fi.JSONName = strings.ToLower(fi.Name)
			}

			si.Fields = append(si.Fields, fi)
		}
		structs = append(structs, si)
		return true
	})

	return structs, nil
}

// FilterByName returns only the structs whose names appear in names.
// Names not found in structs are returned in the missing slice.
func FilterByName(structs []StructInfo, names []string) (matched []StructInfo, missing []string) {
	index := make(map[string]StructInfo, len(structs))
	for _, s := range structs {
		index[s.Name] = s
	}
	for _, name := range names {
		if s, ok := index[name]; ok {
			matched = append(matched, s)
		} else {
			missing = append(missing, name)
		}
	}
	return matched, missing
}

func exprString(expr ast.Expr) string {
	if expr == nil {
		return ""
	}
	switch e := expr.(type) {
	case *ast.Ident:
		return e.Name
	case *ast.SelectorExpr:
		return exprString(e.X) + "." + e.Sel.Name
	case *ast.StarExpr:
		return "*" + exprString(e.X)
	case *ast.ArrayType:
		return "[]" + exprString(e.Elt)
	default:
		return fmt.Sprintf("%T", expr)
	}
}
```

Create `gogen_gen.go` with the generation functions:

```go
package gogen

import (
	"bytes"
	"fmt"
	"go/format"
	"strings"
	"text/template"
)

var constructorFuncs = template.FuncMap{
	"lower": strings.ToLower,
	"requiredFields": func(fields []FieldInfo) []FieldInfo {
		var out []FieldInfo
		for _, f := range fields {
			if f.Required {
				out = append(out, f)
			}
		}
		return out
	},
}

const constructorTmplText = `// Code generated by gogen; DO NOT EDIT.

package {{.Package}}
{{range $s := .Structs}}
// New{{$s.Name}} creates a new {{$s.Name}} with required fields set.
func New{{$s.Name}}({{range $i, $f := requiredFields $s.Fields}}{{if $i}}, {{end}}{{lower $f.Name}} {{$f.Type}}{{end}}) *{{$s.Name}} {
	return &{{$s.Name}}{
		{{- range $s.Fields}}{{if .Required}}
		{{.Name}}: {{lower .Name}},{{end}}{{end}}
	}
}
{{end}}`

const validatorTmplText = `// Code generated by gogen; DO NOT EDIT.

package {{.Package}}

import (
	"fmt"
	"strings"
)
{{range $s := .Structs}}
// Validate checks that all required fields of {{$s.Name}} are set.
func (s *{{$s.Name}}) Validate() error {
	var errs []string
	{{- range $s.Fields}}{{if .Required}}{{if eq .Type "string"}}
	if s.{{.Name}} == "" {
		errs = append(errs, "{{.JSONName}} is required")
	}{{else if eq .Type "int"}}
	if s.{{.Name}} == 0 {
		errs = append(errs, "{{.JSONName}} is required")
	}{{else if .IsPtr}}
	if s.{{.Name}} == nil {
		errs = append(errs, "{{.JSONName}} is required")
	}{{end}}{{end}}{{end}}
	if len(errs) > 0 {
		return fmt.Errorf("validation failed: %s", strings.Join(errs, "; "))
	}
	return nil
}
{{end}}`

type genData struct {
	Package string
	Structs []StructInfo
}

func renderTemplate(name, tmplText string, funcs template.FuncMap, pkg string, structs []StructInfo) ([]byte, error) {
	tmpl, err := template.New(name).Funcs(funcs).Parse(tmplText)
	if err != nil {
		return nil, fmt.Errorf("parse template: %w", err)
	}
	var buf bytes.Buffer
	if err := tmpl.Execute(&buf, genData{Package: pkg, Structs: structs}); err != nil {
		return nil, fmt.Errorf("execute template: %w", err)
	}
	out, err := format.Source(buf.Bytes())
	if err != nil {
		return nil, fmt.Errorf("format: %w\nraw:\n%s", err, buf.String())
	}
	return out, nil
}

// GenerateConstructors generates New<Struct> constructor functions for each struct.
func GenerateConstructors(pkg string, structs []StructInfo) ([]byte, error) {
	return renderTemplate("constructors", constructorTmplText, constructorFuncs, pkg, structs)
}

// GenerateValidators generates Validate() error methods for each struct.
// Only fields tagged validate:"required" are checked.
func GenerateValidators(pkg string, structs []StructInfo) ([]byte, error) {
	return renderTemplate("validators", validatorTmplText, template.FuncMap{}, pkg, structs)
}
```

### Exercise 2: Tests

package gogen

import (
	"strings"
	"testing"
)

var modelSrc = []byte(`package models

import "time"

type User struct {
	ID        int       ` + "`" + `json:"id" validate:"required"` + "`" + `
	Name      string    ` + "`" + `json:"name" validate:"required"` + "`" + `
	Email     string    ` + "`" + `json:"email" validate:"required"` + "`" + `
	Age       int       ` + "`" + `json:"age,omitempty"` + "`" + `
	Bio       string    ` + "`" + `json:"bio,omitempty"` + "`" + `
	CreatedAt time.Time ` + "`" + `json:"created_at"` + "`" + `
	password  string
}

type Config struct {
	Host  string ` + "`" + `json:"host" validate:"required"` + "`" + `
	Port  int    ` + "`" + `json:"port" validate:"required"` + "`" + `
	Debug bool   ` + "`" + `json:"debug,omitempty"` + "`" + `
}
`)

func TestParseStructsFindsExported(t *testing.T) {
	t.Parallel()

	structs, err := ParseStructs(modelSrc)
	if err != nil {
		t.Fatal(err)
	}
	if len(structs) != 2 {
		t.Fatalf("got %d structs, want 2", len(structs))
	}
	names := []string{structs[0].Name, structs[1].Name}
	for _, want := range []string{"User", "Config"} {
		found := false
		for _, n := range names {
			if n == want {
				found = true
			}
		}
		if !found {
			t.Errorf("struct %q not found; got %v", want, names)
		}
	}
}

func TestParseStructsSkipsUnexportedFields(t *testing.T) {
	t.Parallel()

	structs, err := ParseStructs(modelSrc)
	if err != nil {
		t.Fatal(err)
	}
	var user StructInfo
	for _, s := range structs {
		if s.Name == "User" {
			user = s
		}
	}
	for _, f := range user.Fields {
		if f.Name == "password" {
			t.Error("unexported field 'password' should be skipped")
		}
	}
}

func TestParseStructsRequiredTag(t *testing.T) {
	t.Parallel()

	structs, err := ParseStructs(modelSrc)
	if err != nil {
		t.Fatal(err)
	}
	var user StructInfo
	for _, s := range structs {
		if s.Name == "User" {
			user = s
		}
	}
	required := map[string]bool{}
	for _, f := range user.Fields {
		required[f.Name] = f.Required
	}
	if !required["ID"] {
		t.Error("ID should be required")
	}
	if !required["Name"] {
		t.Error("Name should be required")
	}
	if required["Age"] {
		t.Error("Age should not be required (no validate tag)")
	}
	if required["Bio"] {
		t.Error("Bio should not be required (no validate tag)")
	}
}

func TestFilterByName(t *testing.T) {
	t.Parallel()

	structs, err := ParseStructs(modelSrc)
	if err != nil {
		t.Fatal(err)
	}
	matched, missing := FilterByName(structs, []string{"User", "Nonexistent"})
	if len(matched) != 1 || matched[0].Name != "User" {
		t.Errorf("matched = %v, want [User]", matched)
	}
	if len(missing) != 1 || missing[0] != "Nonexistent" {
		t.Errorf("missing = %v, want [Nonexistent]", missing)
	}
}

func TestFilterByNameEmptyNames(t *testing.T) {
	t.Parallel()

	structs, err := ParseStructs(modelSrc)
	if err != nil {
		t.Fatal(err)
	}
	matched, missing := FilterByName(structs, nil)
	if len(matched) != 0 || len(missing) != 0 {
		t.Errorf("FilterByName(nil) = %v, %v; want [], []", matched, missing)
	}
}

func TestGenerateConstructorsProducesValidGo(t *testing.T) {

### Exercise 3: CLI entry point

Create `cmd/gogen/main.go`:

```go
package main

import (
	"flag"
	"fmt"
	"os"
	"strings"

	"example.com/gogen"
)

func main() {
	input := flag.String("input", "", "input Go source file (required)")
	output := flag.String("output", "", "output file (default: <input>_gen.go)")
	typesFlag := flag.String("type", "", "comma-separated struct names to process (default: all)")
	mode := flag.String("mode", "", "generation mode: constructor or validator (required)")
	pkg := flag.String("pkg", "main", "package name for generated file")
	flag.Parse()

	if *input == "" || *mode == "" {
		fmt.Fprintln(os.Stderr, "usage: gogen -input=file.go -mode=constructor|validator [-type=A,B] [-output=out.go] [-pkg=name]")
		os.Exit(1)
	}

	src, err := os.ReadFile(*input)
	if err != nil {
		fmt.Fprintf(os.Stderr, "read %s: %v\n", *input, err)
		os.Exit(1)
	}

	structs, err := gogen.ParseStructs(src)
	if err != nil {
		fmt.Fprintf(os.Stderr, "parse %s: %v\n", *input, err)
		os.Exit(1)
	}

	if *typesFlag != "" {
		names := strings.Split(*typesFlag, ",")
		for i := range names {
			names[i] = strings.TrimSpace(names[i])
		}
		var missing []string
		structs, missing = gogen.FilterByName(structs, names)
		for _, m := range missing {
			fmt.Fprintf(os.Stderr, "warning: type %q not found in %s\n", m, *input)
		}
	}

	if len(structs) == 0 {
		fmt.Fprintln(os.Stderr, "no matching exported structs found")
		os.Exit(1)
	}

	var out []byte
	switch *mode {
	case "constructor":
		out, err = gogen.GenerateConstructors(*pkg, structs)
	case "validator":
		out, err = gogen.GenerateValidators(*pkg, structs)
	default:
		fmt.Fprintf(os.Stderr, "unknown mode %q; use constructor or validator\n", *mode)
		os.Exit(1)
	}
	if err != nil {
		fmt.Fprintf(os.Stderr, "generate: %v\n", err)
		os.Exit(1)
	}

	outPath := *output
	if outPath == "" {
		base := strings.TrimSuffix(*input, ".go")
		outPath = base + "_gen.go"
	}

	if err := os.WriteFile(outPath, out, 0644); err != nil {
		fmt.Fprintf(os.Stderr, "write %s: %v\n", outPath, err)
		os.Exit(1)
	}
	fmt.Printf("generated %s (%d struct(s), mode=%s)\n", outPath, len(structs), *mode)
}
```

## Common Mistakes

### Not separating the model from the AST

Wrong: passing `*ast.TypeSpec` or `*ast.FieldList` directly into the generator template.

What happens: the template is tightly coupled to the AST representation. Testing the generator requires parsing a real file; you cannot feed synthetic data.

Fix: define `StructInfo` and `FieldInfo` as the clean model layer. The parser converts AST nodes to model types; the generator sees only the model.

### Registering template functions after `Parse`

Wrong: calling `tmpl.Funcs(m)` after `template.New("x").Parse(text)`.

What happens: function names are resolved at parse time. Calling `Funcs` after `Parse` has no effect, and execution fails with "function not defined".

Fix: chain `Funcs` before `Parse`: `template.New("x").Funcs(m).Parse(text)`.

### Hard-exiting on unknown type names in `-type`

Wrong: `os.Exit(1)` immediately when `-type=User,Nonexistent` and `Nonexistent` is not in the source.

What happens: the generator aborts even when `User` is valid. During development this forces the caller to fix the type list before any generation proceeds.

Fix: collect missing names, warn on stderr, and continue with the matched subset. Only exit if the matched set is empty.

### Writing directly without formatting

Wrong: writing the raw template output to a file without calling `go/format.Source`.

What happens: the file may contain extra blank lines or subtle syntax errors invisible until `go build` runs.

Fix: always pass template output through `go/format.Source` before writing. On error, log the raw output so you can find the offending line.

## Summary

- Separate parser (AST to model), model (typed structs), and generator (model to `[]byte`) into distinct layers; each is independently testable.
- Parse `validate:"required"` struct tags via `reflect.StructTag(raw).Get("validate")` where `raw` has backticks stripped.
- Use `FilterByName` to match `-type` flag values; warn on missing names but continue with the matched set.
- Register all template functions with `.Funcs(m)` before `.Parse(text)`; function lookup happens at parse time.
- Always format generated output with `go/format.Source`; on failure, log the raw template output to stderr.
